# DesignStudio | ArtStudio | Resume

## Оптимизированный и рефакторенный проект

Этот проект был оптимизирован и рефакторен для улучшения производительности, читаемости кода и архитектуры. Основные улучшения включают:

- Исправление именования компонентов
- Оптимизация загрузки компонентов с использованием lazy loading и Suspense
- Улучшение производительности с помощью requestAnimationFrame в кастомном курсоре
- Исправление MIME-типов для видеофайлов
- Обновление зависимостей до актуальных стабильных версий
- Улучшение архитектуры компонентов

## Установка и запуск

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите проект в режиме разработки:
   ```bash
   npm start
   ```

3. Для сборки проекта:
   ```bash
   npm run build
   ```
